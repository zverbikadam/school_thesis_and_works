# Názov témy

SOAP protokol na výmenu správ založených na XML

# Predmet

Počítačom integrovaná výroba [PIV]

# Typ práce

Semestrálne zadanie

# Rozsah práce

min. 10 strán

## Poznámka

Na zvolenú tému je potrebné vypracovať písomné zadanie s rozsahom minimálne 10 strán, ktoré bude obsahovať prehľad riešenej problematiky a nové trendy v tejto oblasti. V zadaní sa musia nachádzať citácie použitých literárnych zdrojov. Zadanie je možné odovzdať prostredníctvom AIS najneskôr do 19.12.2020 23:59.
